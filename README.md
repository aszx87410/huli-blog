# Blog

## 開發

```
git clone git@github.com:aszx87410/huli-blog.git
npm i
mkdir themes && cd themes
git clone git@github.com:aszx87410/hexo-theme-minos.git
npx hexo s
```

## 部署

```
npx hexo g
npx hexo d
```

## og image

```
npm run og-image -- "redos-regular-expression-denial-of-service"
```
